# 四、系统特点

1. 系统集成json数据处理
2. 工厂模式
3. 操作数据时自动记录操作人员及时间
4. 数据标记删除
5. json数据自动生成历史记录
6. 操作按钮根据权限和Action自动显示隐藏
7. 数据全部缓存，并根据修改过期
8. 数据库自动生成
9. 系统模块基于代码初始化
